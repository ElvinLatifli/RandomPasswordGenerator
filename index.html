MODERN RPG

Modern and useful very strong password generator









Thanks to modern RPG, you can now create very strong passwords! Thanks to the included encryption, you can keep and use your passwords wherever you want!


The application simply opens on the main screen and almost all operations can be done from there. On the screen, there is a bar where the generated password will be displayed, a bar for configuring the password length, and switches for selecting special options that should be included in the password. The progressbar on the side represents the strength of the password. The encryption button opens pages where you can encrypt and decrypt passwords. The tutorial button opens this current page

Sometimes there may be changes in the application interface. Generally, if there is a major change, the tutorial is updated. You need to check the other things by tinkering with them yourself :)













Password length is one of the most important issues in terms of security. With RPG, you can create a password with a minimum length of 7 characters and a maximum length of 36 characters.
Why 36? Because at 36 characters it is long enough and quite safe. We didn't need any more length.
Why at least 7? Since even a 7-character password is weak enough, it doesn't make sense to create a password with fewer characters!

















Here you can choose the features that will be included in your password with switch cases. uppercase letters, numbers and special characters. All of them are turned on by default to ensure top-level security, but you can configure them as you wish. If you turn off all of them, a password consisting of only lowercase letters will be created. If you want high level security it is important to open them all!







To create the password, it is important to click on the "Generate password" button below after making your selections. We implemented all high-level security measures for testing purposes and activated the maximum password length and all selections. As you can see, our progress bar shows that the password is very secure.




















ENCRYPTION

Keep the passwords you created

wherever you want by re-encrypting them!

Since it is almost impossible to remember the created password, we offer you such a feature within the application. If you want to keep the password you created in this way somewhere, you can encrypt it again and save it in the txt file you want. At the same time, when you use the password, you can decrypt it from the decryption section in the application. However, when decrypting, do not forget to give the decryptor a name and a "simple" password. So when you save it, it will create a txt file on your desktop and print the necessary information there! Thus, even if your txt file is accessed without permission, it will not be able to learn your real password.
